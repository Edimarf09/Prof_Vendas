## Overview

This project is a web platform built with **Django** for selling and distributing educational materials created by teachers.

The idea is to provide a marketplace where each user can create an account, manage their own products, and sell teaching resources such as worksheets, classroom activities, lesson materials, and other digital educational content.

This repository represents the development of a scalable backend-oriented system designed with real deployment in mind.

## Purpose

This project was created as part of my personal portfolio.

Its purpose is to demonstrate:

- backend development with Django
- marketplace-oriented system design
- project organization
- preparation for VPS deployment
- use of containerized services during development and production

## Tech Stack

Current and planned technologies include:

- **Django** — backend framework
- **MySQL** — relational database
- **Docker / Docker Compose** — containerized services and deployment
- **PagSeguro / PagBank API** — payment integration
- **Gunicorn + Nginx** — production deployment stack

Additional tools and services may be introduced as the project evolves.

## Project Goals

The platform is intended to support features such as:

- user authentication and account creation
- seller account management
- product registration and organization
- digital product sales
- payment processing
- order and transaction management
- administrative controls
