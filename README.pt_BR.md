## Visão Geral

Este projeto é uma plataforma web construída com **Django** para a venda e distribuição de materiais educacionais criados por professores.

A ideia é fornecer um marketplace onde cada usuário possa criar uma conta, gerenciar seus próprios produtos e vender recursos pedagógicos, como folhas de atividades, atividades para sala de aula, materiais de aula e outros conteúdos educacionais digitais.

Este repositório representa o desenvolvimento de um sistema escalável, orientado a backend, projetado com foco em implantação real.

## Propósito

Este projeto foi criado como parte do meu portfólio pessoal.

Seu objetivo é demonstrar:

- desenvolvimento backend com Django
- design de sistemas orientados a marketplace
- organização de projeto
- preparação para implantação em VPS
- uso de serviços conteinerizados durante o desenvolvimento e a produção

## Stack de Tecnologias

As tecnologias atuais e planejadas incluem:

- **Django** — framework backend
- **MySQL** — banco de dados relacional
- **Docker / Docker Compose** — serviços conteinerizados e implantação
- **PagSeguro / PagBank API** — integração de pagamentos
- **Gunicorn + Nginx** — stack de implantação em produção

Ferramentas e serviços adicionais poderão ser introduzidos conforme o projeto evoluir.

## Objetivos do Projeto

A plataforma foi projetada para oferecer suporte a funcionalidades como:

- autenticação de usuários e criação de contas
- gerenciamento de contas de vendedores
- cadastro e organização de produtos
- venda de produtos digitais
- processamento de pagamentos
- gerenciamento de pedidos e transações
- controles administrativos
